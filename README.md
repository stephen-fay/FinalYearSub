# Ionic 2 Leaflet Map Example

Ionic 2 Example App with Leaflet using OpenStreetMap Layers.

## how to continue

	git clone https://github.com/SBejga/ionic2-map-leaflet
	cd ionic2-map-leaflet
	npm install
